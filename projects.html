✅ Project Description: Do-It!

Do-It! is a clean, responsive to-do list app designed to help you organize your day, stay focused, and boost productivity. Whether you're managing personal goals, tasks, or simple reminders, Do-It! offers an intuitive interface that works seamlessly across devices.

Built with HTML, CSS, and JavaScript, this app allows users to:

📝 Add and delete tasks with a single click
✅ Mark completed tasks
📱 Use the app comfortably on both desktop and mobile screens

🌙 Experience a smooth, minimal design with responsive layout adjustments

With Do-It!, staying organized is just one click away. No sign-ups, no distractions — just your goals, clearly laid out.

✅ Project Description: QuickQuiz

QuickQuiz is a simple, responsive quiz app designed to test your knowledge on various topics. With an intuitive interface and engaging questions, it provides a fun way to learn and challenge yourself.

Built with HTML, CSS, and JavaScript, this app allows users to:

❓ Take quizzes on different subjects
📝 Submit answers and receive instant feedback
📱 Enjoy a mobile-friendly design for quizzes on the go

🌟 Experience a clean layout with smooth transitions between questions

QuickQuiz makes learning enjoyable and accessible. No sign-ups required — just pick a quiz and start testing your knowledge!

✅ Project Description: WeathR

WeathR is a lightweight, real-time weather app that lets users quickly check current weather conditions for any city worldwide. Built for simplicity and speed, WeathR delivers accurate forecasts through a clean, mobile-friendly interface.

Key Features:

🌐 Global city search for weather data
🌡️ Real-time temperature, humidity, and conditions
📱 Responsive design for seamless mobile use

⚡ Fast loading with no installation required

With WeathR, you get the weather you need — no clutter, just clarity.
